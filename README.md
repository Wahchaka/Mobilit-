Pour lancer l'application :
1. Double-cliquez sur "backend.exe"
2. Une fenêtre noire s'ouvre (ne pas la fermer)
3. Le navigateur s'ouvre automatiquement
4. Remplissez la fiche et validez

Le fichier Excel se génère automatiquement sur votre Bureau.

Pour fermer l'application : fermez la fenêtre noire.
